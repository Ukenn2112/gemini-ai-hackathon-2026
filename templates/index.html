🚀 Cloud Run 部署指南

您已经准备好最简单的 Hello World 结构。可以通过以下步骤将此应用发布到 Google Cloud Run：

  • 1
    使用 MCP 工具部署

    在当前 Agent 对话中使用 mcp_cloudrun_deploy_local_folder 部署当前文件夹。

  • 2
    绑定 API Key 变量

    部署时设置环境变量：
    GEMINI_API_KEY=您的Key

若要在本地测试，可运行以下命令：

# 安装依赖

# 设置环境变量
set GEMINI_API_KEY="您的API_KEY"

# 启动服务
python app.py
